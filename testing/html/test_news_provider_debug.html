Error Analysis
The error "useNewsContext must be used within a NewsProvider" indicates that components are trying to use the context but it's not available.
Error: useNewsContext must be used within a NewsProvider
Location: NewsContextNew.tsx:161:11
Components affected: NewsLeftCardWrapper, NewsRightCard
Possible Causes
Common reasons why this error occurs:
1. Provider Not Wrapping Components: NewsProvider is not properly wrapping the components that use useNewsContext
2. Provider Failing to Render: NewsProvider is failing to render due to GraphQL errors or other issues
3. Timing Issues: Components are rendering before the provider is ready
4. Import Issues: Wrong import path or circular dependencies
5. Context Not Provided: NewsContext.Provider is not providing the context value
